This CRUD app is designed to manage a set of users who each may belong to one group.

A proper .csv will contain a set of users who should have the following characteristics:

  • first name
  • last name
  • address
  • city
  • state, ie. NY, TX
  • user number
  • group: name, symbol, ie. Murphyverse, MRPH
