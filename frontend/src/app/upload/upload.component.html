This CRUD app is designed to manage a set of users who each may belong to one group.

A proper .csv will contain a set of users who should have data which corresponds with these headers:

  • 'firstname'
  • 'lastname'
  • 'address'
  • 'city'
  • 'state', ie. NY
  • 'usernumber'
  • 'group', ie. '1', which corresponds with group 1 in the group database

If you submit a .csv with improper headers, the data will not be posted to the backend. Otherwise, your data will be accepted. This application is run on your own local database and only has access to data that you give it access to.
