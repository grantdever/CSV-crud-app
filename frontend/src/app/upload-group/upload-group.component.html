Let's upload a group file!

This app is designed to manage a set of users, who each may belong to one group.

By uploading an .csv below, you can add a group (or multiple) to the database.

A proper .csv will contain a group, or series of group, with data which corresponds with these headers:

  • 'name'
  • 'symbol'

If you submit a .csv with improper headers, the data will not be posted to the backend. Otherwise, your data will be accepted. This application is run on your own local database and only has access to data that you give it access to.
