Welcome to my first CRUD application!

The upload page upload will:

  • allow the user to upload an .csv file
  • validate that it is an .csv
  • return any errors

If the .csv file is properly formatted for my application:

  • this data will be converted to JSON
  • added to my database
  • rendered in the appropriate frontend views.

This is my first full stack project. It is built with: Angular, PostgreSQL, TypeORM, and NestJS.
