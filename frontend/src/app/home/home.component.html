Welcome to my first CRUD application!

The upload page:

  • enables users to upload an .csv file
  • validates that they are uploading an .csv
  • validates that the headers are correct

If the .csv file contains the proper headers and the db is configured:

  • this data will stored in a variable
  • this data will be passed into a POST end point and the data will be added to the database
  • the user will be notified that this has successfully occurred
  • the user can navigate to the appropriate routes to view this data

This is my first full stack project. It is built with: Angular, PostgreSQL, TypeORM, and NestJS.
